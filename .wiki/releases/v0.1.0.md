# v0.1.0 Release Notes

**Release Date**: January 1, 2025

The initial release of nself - Self-Hosted Infrastructure Manager.

First public release establishing the foundation for nself as a Docker-based development environment manager.

- Initial nself CLI functionality

- Core command structure
- Environment file management
