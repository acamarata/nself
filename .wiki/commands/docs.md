# nself docs

Open documentation URLs in browser or print URL targets.

## Status

- Runtime command exists (`src/cli/docs.sh`).
- Canonical v1 organization maps docs tooling under `nself dev docs`.

## Syntax

```bash
nself docs [options] [section]
```

## Options

- `-h, --help`
- `-u, --url-only`
- `-w, --wiki`
- `-r, --readme`
- `-g, --guides`
- `-a, --api`
- `-c, --commands`
