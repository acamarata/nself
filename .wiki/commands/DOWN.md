# nself down - Stop Services (Legacy)

**Version 0.9.8** | Stop all services

---

## Overview

The `nself down` command stops all nself services. This is a legacy alias for `nself stop`.

---

## Usage

```bash
# Stop all services
nself down

# Equivalent to:
nself stop
```

---

## Note

For new projects, use `nself stop` instead. The `down` command is maintained for backward compatibility.
