# nself v1.x Roadmap

Post-v1.0 feature roadmap for the v1.x series.

**Timeframe**: Q2 2026 - Q4 2027 (18 months)
**Philosophy**: Incremental improvements, no breaking changes

| Version | Target | Focus | Duration |

## v1.1.0 - Plugin Marketplace (Q2 2026)

### Core Features
- Plugin marketplace website
- Plugin discovery and search
- Plugin ratings and reviews
- Automated plugin installation
- Plugin dependency management
- Plugin updates and versioning

### Official Plugins
- **Stripe Plugin** - Billing/payments sync
- **GitHub Plugin** - Repository/issue sync
- **Shopify Plugin** - E-commerce integration
- **Twilio Plugin** - SMS/voice integration
- **SendGrid Plugin** - Email tracking

### Additional Migrations
- Heroku → nself migration
- AWS Amplify → nself migration
- Parse Server → nself migration

### Developer Tools
- Plugin SDK improvements
- Plugin testing framework
- Plugin documentation generator
- Example plugin templates
