## Quick Start

### GraphQL API

```bash
# Access GraphQL console
open https://api.local.nself.org

# Generate TypeScript types
nself db types
```

### Authentication

```bash
# Get auth endpoints
nself urls auth

# Test authentication
curl https://auth.local.nself.org/healthz
```
