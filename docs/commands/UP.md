# nself up - Start Services (Legacy)

**Version 0.4.8** | Start all services

---

## Overview

The `nself up` command starts all nself services. This is a legacy alias for `nself start`.

---

## Usage

```bash
# Start all services
nself up

# Equivalent to:
nself start
```

---

## Note

For new projects, use `nself start` instead. The `up` command is maintained for backward compatibility.
